Registrar

Para acceder a la aplicación, tenemos que tener un usuario registrado, por lo tanto, lo primero que haremos es registrarnos. Desde la pantalla inicial de la app (Login), pinchamos en ‘Registrar’, una vez estamos en la pantalla de registro tendremos que rellenar nombre, usuario, contraseña y edad. La imagen no es obligatoria.

Login

Una vez registrado, ya podremos acceder a la aplicación con ese usuario, logueandonos. En la pantalla inicial, introducimos usuario y contraseña y pulsamos ‘Login’. Con esto, si existe el usuario, accederemos a la aplicación y la podremos usar como un usuario más.

Perfil

Dentro de la aplicación podremos editar el usuario. Tanto los datos que hemos introducido en el registro, como las necesidades y qué elementos debe mostrar la aplicación al usuario. Al registrarnos, por defecto, el usuario se crea sin necesidades y para que se le muestre todos los elementos de accesibilidad.
